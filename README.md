# 🚲 Bikeshare Analysis

## Overview
Analyze bikeshare data to explore usage patterns by city, month, and day.

## Features
- Filter by city, month, day  
- Stats: time, stations, trips, duration, users  
- Optional raw data display  

## Setup
```bash
git clone https://github.com/your-username/bikeshare-project.git
cd bikeshare-project
pip install -r requirements.txt
```

### Run
```python bikeshare.py```

### Tech
Python, Pandas, NumPy
